Instalador padrão de softwares para Windows. Instala aplicativos via WinGet, realiza otimizações do sistema e gerencia usuários locais. Protegido por senha.

Windows 10/11 WinGet Administrador PowerShell
Log em C:\Temp\Instalador-Padrao.log
Baixar .exe

O Windows Defender pode exibir um aviso na primeira execução. Clique em "Mais informações" → "Executar assim mesmo" para prosseguir.

Como usar

1

Baixe o arquivo

2

Execute como Admin

Clique com botão direito → Executar como administrador.

3

Digite a senha

Informe a senha de acesso fornecida pelo analista de TI.

4

Use os menus

Navegue pelas opções e o log é salvo automaticamente.
