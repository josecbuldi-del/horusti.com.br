Como usar
01

Abra o PowerShell como Administrador

Clique com o botão direito no menu Iniciar e escolha Windows PowerShell (Admin) ou Terminal (Admin).

02

Cole e execute o comando

Copie o comando acima, cole no terminal e pressione Enter. O script será baixado e executado automaticamente.

03

Siga as instruções na tela

O instalador irá guiar você pelo processo. Confirme as permissões quando solicitado.
