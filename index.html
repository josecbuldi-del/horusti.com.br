Windows Defender: ao executar pela primeira vez o Windows pode exibir um aviso de segurança. Clique em "Mais informações" → "Executar assim mesmo" para prosseguir. Isso ocorre pois o executável não possui assinatura digital.

Como usar

01

Baixe o executável

Clique no botão acima para baixar o arquivo Instalar-Padrao.exe para o seu computador.

02

Execute como Administrador

Clique com o botão direito no arquivo e selecione Executar como administrador. O programa já solicita elevação automaticamente ao abrir.

03

Informe a senha de acesso

O instalador solicita autenticação antes de liberar os menus. Entre em contato com o analista de TI responsável para obter a senha.

04

Navegue pelos menus

Use as opções numeradas para instalar aplicativos, otimizar o sistema ou gerenciar usuários. O log é gerado automaticamente em C:\Temp\Instalador-Padrao.log.

Módulo 01

Instalar Apps

Chrome, Adobe Reader, WinRAR e 7-Zip via WinGet silencioso.

Módulo 02

Otimização

Limpeza, SFC, DISM, Prefetch, DNS, RAM, desfrag e muito mais.

Módulo 03

Usuários

Criar, remover, trocar senhas e gerenciar grupos locais.

Segurança

Acesso Restrito

Autenticação por hash SHA-256. Histórico apagado ao sair.
